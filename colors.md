## Light Mode

| Hex Code | Name | Usage |
|----------|------|-------|
| `#ec4755` | LOME red | Primary color, all heading colors (h1-h6), AppBar gradient (left side) |
| `#7209b7` | Vivid purple | Secondary color |
| `#ffba75` | Orange accent | Tertiary color, AppBar gradient (right side) |
| `#ffba7544` | Orange (translucent) | Tertiary light variant |
| `#fef8e0` | Warm cream | Background default, body background |
| `#ffffff` | White | Paper/card backgrounds |
| `#000000` | Black | Primary text |
| `#444444` | Dark gray | Secondary text |
| `#4CC9F0` | Neon blue | Accent color |
| `#FF5757` | Red | Error states |
| `#FFC914` | Yellow | Warning states |
| `#00BBF9` | Sky blue | Info states |
| `#00F5D4` | Neon green | Success states |
